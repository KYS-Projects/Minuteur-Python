# Minuteur-Python

Application de **minuteur avec interface graphique** développée en **Python avec Pygame**.  
Le programme permet de configurer un **compte à rebours en heures, minutes et secondes**, avec des **alertes sonores à des intervalles spécifiques** et une **notification finale** lorsque le temps est écoulé.

## Fonctionnalités

- Interface graphique interactive avec **Pygame**
- Réglage du temps (heures, minutes, secondes)
- Bouton **Démarrer / Pause**
- Bouton **Annuler / Réinitialiser**
- Compte à rebours en temps réel
- Alertes sonores automatiques à :
  - 15 minutes
  - 10 minutes
  - 5 minutes
  - 2 minutes
  - 1 minute
- Son final lorsque le minuteur atteint **00:00:00**

## Technologies utilisées

- **Python**
- **Pygame** (bibliothèque pour créer des applications graphiques et gérer les événements)

## Environnement requis

Pour exécuter ce projet, vous devez avoir :

- **Python 3.x** installé sur votre ordinateur
- Un **terminal ou invite de commande**
- Un éditeur de code comme :
  - VS Code
  - PyCharm
  - ou tout autre éditeur Python

## Installation : exécuter la commande suivante dans le terminal

1. Cloner le dépôt

Télécharger le projet depuis GitHub :

 ```bash
  git clone https://github.com/KYS-Projects/Minuteur-Python.git
   ```
2. Accéder au dossier du projet
   
cd Minuteur-Python

4. Installer les dépendances
Installer les bibliothèques nécessaires :

 ```bash
pip install -r requirements.txt
   ```
Lancer l'application

Pour démarrer l'application, exécuter la commande suivante dans le terminal :

 ```bash
python main.py
   ```
Une fenêtre graphique du minuteur s’ouvrira et vous pourrez configurer la durée du compte à rebours.
